Auralethi

Projects

Auralethi is developing several connected initiatives. Each project represents one layer of a larger system.

The organization is currently in Phase 1 — Foundation, so most projects are in design or planning.

Phase 1 — Design
Hollarin AI Companion

Human-centered AI support for memory, organization, and daily life.

Planned features:

  • task guidance
  • reminders
  • executive function support
  • structured assistance
  • ethical AI constraints

Prototype work will begin after the architecture is stable.

Phase 1 — Internal architecture
CortexOS Architecture

CortexOS is the internal architecture being designed to support future AI tools.

It is not a product. It is the framework used to build systems like the Hollarin AI Companion.

Current work includes:

  • layered architecture design
  • testing AI support models
  • defining constraints
  • documentation
Phase 1 — Early development
KEFR Radio

KEFR Radio is a small public media channel used to share project updates, music, ideas, and development logs while the organization is still in early phases.
